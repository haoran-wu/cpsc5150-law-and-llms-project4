# Project 4

This folder contains the final Project 4 submission on translating 26 U.S.C. §152 into SMT and checking scenarios with Z3.

Important files:

- `Project 4.pdf`: assignment prompt
- `report.md`: submission report
- `smt/`: SMT-LIB files
- `scenarios/`: scenario descriptions
- `solver_outputs/`: Z3 outputs
- `part_b/`: Part B scenario and LLM comparison materials
- `scripts/`: development prompts and helper script
- `transcripts/`: iteration notes

This project already has a Git repository and is ready to publish or submit from the `main` branch.
